# Contributing to SHOFTY

Contributions are welcome! SHOFTY is a learning project, so beginner PRs are totally fine.

## How to contribute
1. Fork the repo
2. Create a branch for your change
3. Test it: `make run` should boot to the shell
4. Open a Pull Request explaining what you changed

## Rules
- only code languages and tools
- Comments in English
- Keep main bootable
